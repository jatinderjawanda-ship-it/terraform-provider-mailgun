# mailgun_domain_sending_key (Resource)

Creates a domain-scoped sending API key for Mailgun. This key can only be used to send emails via the specified domain. Use this resource to provision application credentials for sending email.

~> **Important:** The API key secret is only available immediately after creation and cannot be retrieved later. Store the secret securely (e.g., in HashiCorp Vault) as soon as the resource is created.

## Example Usage

```terraform
# Create a domain sending key for an application
resource "mailgun_domain_sending_key" "app" {
  domain      = "mail.example.com"
  description = "Production application key"
}

# Store the secret in Vault
resource "vault_generic_secret" "mailgun_key" {
  path = "secret/mailgun/app"

  data_json = jsonencode({
    api_key = mailgun_domain_sending_key.app.secret
  })
}

# Create a sending key with expiration
resource "mailgun_domain_sending_key" "temporary" {
  domain      = "mail.example.com"
  description = "Temporary key for migration"
  expiration  = 86400 # 24 hours in seconds
}

output "sending_key_id" {
  value = mailgun_domain_sending_key.app.id
}

# Note: The secret is only available after creation and cannot be retrieved later.
# Make sure to store it securely (e.g., in HashiCorp Vault) immediately.
output "sending_key_secret" {
  value     = mailgun_domain_sending_key.app.secret
  sensitive = true
}
```

## Schema

### Required

- `domain` (String) The domain this sending key is scoped to. The key can only send emails from this domain.

### Optional

- `description` (String) A human-readable description for this API key (e.g., 'Production app key').
- `expiration` (Number) The key's lifetime in seconds. Set to 0 (default) for no expiration.

### Read-Only

- `created_at` (String) The timestamp when this API key was created.
- `expires_at` (String) The timestamp when this API key expires. Empty if the key does not expire.
- `id` (String) The unique identifier for this API key.
- `secret` (String, Sensitive) The API key secret. This is only available immediately after creation and cannot be retrieved later. Store this value securely (e.g., in HashiCorp Vault) as it will not be shown again.

## Import

Import is supported using the following syntax:

```shell
# Import a domain sending key using its ID
# Note: The secret cannot be retrieved after import
terraform import mailgun_domain_sending_key.app key-1234567890abcdef
```

~> **Note:** The secret cannot be retrieved when importing an existing key. If you need the secret, create a new key instead.
