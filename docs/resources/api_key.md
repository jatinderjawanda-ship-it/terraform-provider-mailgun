# mailgun_api_key (Resource)

Manages a Mailgun API key. API keys provide programmatic access to Mailgun services.

## Example Usage

```terraform
# Create a domain-specific sending key
resource "mailgun_api_key" "sending" {
  role        = "sending"
  kind        = "domain"
  domain_name = "mail.example.com"
  description = "Application sending key"
}

# Create an admin key (use with caution)
resource "mailgun_api_key" "admin" {
  role        = "admin"
  description = "Admin API key for management"
}

# The secret is only available after creation
output "api_key_secret" {
  value     = mailgun_api_key.sending.secret
  sensitive = true
}
```

## Schema

### Required

- `role` (String) The role for this API key. Valid values: 'admin', 'sending', 'developer', 'basic', 'support'. For domain-specific sending keys, use 'sending' with 'kind' set to 'domain'.

### Optional

- `description` (String) A human-readable description for this API key.
- `domain_name` (String) The domain this API key is scoped to. Required when 'role' is 'sending' and 'kind' is 'domain'.
- `expiration` (Number) The key's lifetime in seconds. Set to 0 for no expiration. Web keys have a maximum lifetime of 1 day.
- `kind` (String) The type of API key. Valid values: 'domain' (for domain-specific sending keys), 'user' (default, for user-level keys), 'web' (for web keys with 1-day max lifetime).

### Read-Only

- `created_at` (String) The timestamp when this API key was created.
- `disabled_reason` (String) The reason this API key was disabled, if applicable.
- `expires_at` (String) The timestamp when this API key expires. Empty if the key does not expire.
- `id` (String) The unique identifier for this API key.
- `is_disabled` (Boolean) Whether this API key is disabled.
- `requestor` (String) The entity that requested this API key.
- `secret` (String, Sensitive) The API key secret. This is only available immediately after creation and cannot be retrieved later. Store this value securely as it will not be shown again.
- `updated_at` (String) The timestamp when this API key was last updated.
- `user_name` (String) The username associated with this API key.
